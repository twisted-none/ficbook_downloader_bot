# Ficbook Telegram Bot

Telegram-бот на `aiogram`, который принимает ссылку на Ficbook, постранично загружает текст через `FanFicFare`, собирает `.fb2` и отправляет файл в ответ.

## Возможности

- принимает ссылки `ficbook.net` и `ficbook.com`
- скачивает многоглавные фанфики по главам
- собирает итоговый `fb2` в памяти
- поддерживает авторизацию для `18+` через `FICBOOK_LOGIN` и `FICBOOK_PASSWORD`
- готов для запуска в Docker

## Запуск

```bash
cp .env.example .env
docker compose up --build -d
```

## Переменные окружения

- `BOT_TOKEN` — токен Telegram-бота
- `FICBOOK_LOGIN` — логин Ficbook для `18+`
- `FICBOOK_PASSWORD` — пароль Ficbook для `18+`
- `ADMIN_CHAT_ID` — ваш `chat_id` для уведомлений об ошибках пользователей
- `LOG_LEVEL` — уровень логирования, по умолчанию `INFO`

## Поведение

Бот отвечает на сообщение со ссылкой на фанфик и отправляет `.fb2`. Если фик `18+`, но логин или пароль не заданы, бот вернет понятную ошибку вместо падения. Если задан `ADMIN_CHAT_ID`, бот дополнительно пришлет вам сообщение с данными пользователя и текстом ошибки.
